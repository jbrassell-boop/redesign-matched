# Design Context

## Target Audience
Internal staff at Total Scope Inc. (TSI), a medical device repair company. Users are operations managers, technicians, sales reps, and administrative staff who process repair work orders, track inventory, manage contracts, and generate reports. Users are power users who spend all day in this app.

## Use Cases
- Create and manage repair work orders (flexible/rigid endoscopes, surgical instruments)
- Track repair status, technician assignments, and turnaround times
- Manage clients, departments, contracts, and pricing
- Run operational dashboards and financial reports
- Process receiving, shipping, and inventory
- Quality control and inspection workflows

## Brand Personality / Tone
- **Professional and utilitarian** — this is a workhorse app, not a marketing site
- **Data-dense** — users need information density, not whitespace
- **Efficient** — minimize clicks, maximize throughput
- **Trustworthy** — medical device industry, accuracy matters
- **Clean but not flashy** — Ant Design component library, blue primary color (#1677ff)

## Technical Stack
- React 18 + TypeScript
- Ant Design (antd) component library
- ASP.NET Core backend API
- SQL Server database

## Design System
- Ant Design default theme with minimal customization
- Standard antd components: Tables, Forms, Modals, Tabs, Drawers
- Dense table layouts with inline actions
- Tab-based detail panes for record views

## What This Is NOT
- Not a consumer product — no onboarding flows, no marketing pages
- Not mobile-first — desktop-primary internal tool
- Not a design showcase — function over form
